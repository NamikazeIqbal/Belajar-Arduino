Perintah dikirim dari komputer ke arduino simulator di proteus menggunakan Virtual Serial Port Emulator (VSPE), serta Hype!Terminal untuk
mengetik perintah dengan baudrate 9600. Sehingga, apabila ingin menggunakan kode ini, install terlebih dahulu VSPE, dan Hype!Terminal.

Untuk menghubungkan COMPIM dengan virtual server, atur COMPIM agar mengambil port virtual yang telah dibuat menggunakan VSPE, dan atur
baudrate sama dengan pengaturan pada Hype!Terminal. Selanjutnya, upload code ke masing-masing arduino.
